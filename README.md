# Portfolio — Bárbara Soto
**Directora de Arte**

Portfolio personal desarrollado en HTML/CSS/JS puro.

## Tecnologías
- HTML5 / CSS3 / JavaScript vanilla
- Three.js r128 (objeto 3D interactivo)
- Sin dependencias externas (todo embebido)

## Cómo usar
1. Abre la carpeta en VS Code
2. Instala la extensión **Live Server**
3. Click derecho en `index.html` → **Open with Live Server**

## Deploy en Vercel
1. Sube esta carpeta a un repositorio de GitHub
2. Entra a [vercel.com](https://vercel.com) y conecta el repo
3. Vercel detecta automáticamente el proyecto estático
4. Deploy automático ✓
